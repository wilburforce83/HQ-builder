# ![Alt text](card-builder/public/static/img/ui/logo.png "HeroQuest Quest Builder")

## Overview
This repo is a single **Next.js (Node + React)** app located in `card-builder/`.
- Quest Builder: `/`
- Card Builder: `/cards`
- API: `/api/*`

See `BUILD_AND_TEST.md` for setup and build instructions.

## Data
All data is stored in SQLite at:
```
db/hqbuilder.db
```
